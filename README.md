# SKYRA Jewels - Full Stack Jewelry E-Commerce Website

SKYRA Jewels is a full-stack jewelry e-commerce platform built for a real jewelry brand.

Live Website: https://skyrajewels.co.in

## Features

- User registration and login
- Google authentication
- User profile
- Dynamic product listing
- Product detail page
- Cart and wishlist
- Checkout flow
- Razorpay payment integration
- Order management
- Admin dashboard
- Product and category management
- Cloudinary image upload
- Email order notifications
- Responsive design

## Tech Stack

### Frontend
- React
- Vite
- React Router
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### Services
- Firebase Authentication
- Cloudinary
- Razorpay
- Brevo

## Admin Features

- Add, edit, and manage products
- Add, edit, and manage categories
- Manage orders
- Update order status
- Manage shipping settings
- Manage admin users
